Install in under a minute

Not on the Chrome Web Store yet — load it as an unpacked extension.

1

Download the source

Clone or download this repository from GitHub.

2

Open Chrome extensions page

Go to chrome://extensions and turn on Developer mode (top right).

3

Load unpacked

Click Load unpacked and select the project folder (the one containing manifest.json).

4

Configure your channels

Click the extension icon, set your default skip second, and add the channels you want. Done!

FAQ

Does this skip intros on every channel?

No. It only acts on channels you've explicitly added in the popup. Everything else plays normally.

Will I see the intro for a split second before it skips?

The extension pauses playback the instant it starts (before the first frame renders), figures out the channel, then seeks and resumes — so there's no visible flash in normal use.

Does it send my data anywhere?

No. Everything runs locally in your browser and settings are stored with Chrome's built-in storage.sync API.

Does it work on Shorts and normal videos?

Yes, both are supported since the extension reads the video ID from the URL for either format.
